PHP para Principiantes

1. ¿Qué es PHP?

PHP (Hypertext Preprocessor) es un lenguaje de programación de propósito general que se utiliza principalmente para el desarrollo de aplicaciones web. PHP se ejecuta en el servidor y es capaz de generar contenido dinámico en páginas web.

2. ¿Cómo Empezar con PHP?

Para empezar a usar PHP, necesitas instalar un servidor local, como XAMPP o WAMP. Esto te permitirá ejecutar archivos PHP en tu computadora sin necesidad de subirlos a un servidor en línea.

Paso 1: Descarga e instala XAMPP o WAMP.

Paso 2: Crea un archivo PHP en la carpeta de tu servidor local.

Paso 3: Escribe el código PHP en el archivo y guárdalo con la extensión .php.

3. Escribir el Primer Código en PHP

Veamos cómo crear un simple programa en PHP que muestra "¡Hola, Mundo!" en la pantalla.

<?php
echo "¡Hola, Mundo!";
?>

Guarda este código en un archivo llamado index.php y ejecútalo en el navegador para ver el mensaje.

4. Variables en PHP

En PHP, las variables son contenedores que almacenan información. Se inician con el símbolo $ seguido del nombre de la variable.

Ejemplo:

<?php
$nombre = "Javier";
$edad = 25;
echo "Mi nombre es " . $nombre . " y tengo " . $edad . " años.";
?>

Este código mostrará "Mi nombre es Javier y tengo 25 años."

5. Estructuras de Control

Las estructuras de control, como condicionales y bucles, te permiten controlar el flujo de tu programa.

Condicional if

<?php
$edad = 18;
if ($edad >= 18) {
    echo "Eres mayor de edad.";
} else {
    echo "Eres menor de edad.";
}
?>

Este código verifica si la variable $edad es mayor o igual a 18.

6. Bucles en PHP

Los bucles permiten ejecutar un bloque de código varias veces. PHP ofrece varios tipos de bucles: while, for, foreach, entre otros.

Bucle for

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Número: " . $i . "<br>";
}
?>

Este código imprime los números del 1 al 5.

7. Funciones en PHP

Las funciones son bloques de código que se pueden reutilizar. PHP tiene muchas funciones integradas y también puedes crear las tuyas.

Ejemplo de una función simple:

<?php
function saludo($nombre) {
    return "¡Hola, " . $nombre . "!";
}
echo saludo("Javier");
?>

Este código mostrará "¡Hola, Javier!".

8. Conclusión

En este tutorial básico, aprendiste los fundamentos de PHP, incluyendo variables, estructuras de control, bucles y funciones. Con esta base, ya puedes empezar a construir aplicaciones web dinámicas. ¡Sigue practicando para dominar PHP!
