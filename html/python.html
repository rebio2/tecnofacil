Introducción a Python

Python es un lenguaje de programación versátil y fácil de aprender. Es utilizado en una amplia variedad de aplicaciones, desde desarrollo web hasta ciencia de datos e inteligencia artificial.

¿Qué es Python?

Python es un lenguaje de programación de alto nivel creado por Guido van Rossum en 1991. Es conocido por su sintaxis simple y su facilidad de uso, lo que lo convierte en una excelente opción para principiantes.

Instalación de Python

Para empezar a programar en Python, primero necesitas instalarlo en tu sistema. Puedes descargarlo desde el sitio web oficial de Python en python.org.

# En la terminal o consola
python --version

Este comando te permite verificar si Python se ha instalado correctamente.

Primer Programa: "Hola, Mundo"

El siguiente código muestra cómo imprimir un mensaje en pantalla usando Python:

print("Hola, Mundo")

Al ejecutar este código, deberías ver la frase "Hola, Mundo" en la consola.

Variables y Tipos de Datos

En Python, puedes almacenar información en variables. A continuación se presentan algunos tipos de datos comunes:

  • Cadena de texto (String): texto rodeado por comillas. Ejemplo: nombre = "Juan"
  • Números enteros (Int): números sin decimales. Ejemplo: edad = 25
  • Números decimales (Float): números con decimales. Ejemplo: altura = 1.75
  • Booleano (Bool): valores de True o False. Ejemplo: es_estudiante = True
nombre = "Juan"
edad = 25
altura = 1.75
es_estudiante = True

Estructuras de Control

Las estructuras de control permiten que tu código tome decisiones. Aquí te mostramos ejemplos de una estructura condicional if y un bucle for:

# Estructura condicional
edad = 18
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

# Bucle for
for i in range(5):
    print(i)

En este ejemplo, el código verifica si una persona es mayor de edad y luego usa un bucle para contar de 0 a 4.

Funciones

Las funciones son bloques de código reutilizables. Puedes definir una función en Python usando la palabra clave def:

def saludar(nombre):
    print("Hola, " + nombre)

saludar("Ana")

Este código define una función llamada saludar que imprime un saludo personalizado.

Conclusión

¡Felicidades! Has aprendido los conceptos básicos de Python. Con estos fundamentos, estás listo para comenzar a explorar proyectos más complejos y profundizar en el mundo de la programación.
